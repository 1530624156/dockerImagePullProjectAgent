Docker Images Pull Proxy

通过代理拉取 Docker 镜像并导出为 tar 包

使用方法

1
下载 tar 包

在 镜像列表 中找到拉取完成的镜像，点击下载对应的 tar 包。

2
加载镜像

将 tar 包上传到目标服务器，执行命令加载镜像：

$ docker load -i xxx.tar
3
验证镜像

查看镜像是否已成功加载：

$ docker images
